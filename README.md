# gns3-cisco-3725-bgp-dual-isp-lab

What this lab is - Dual-ISP eBGP lab in GNS3 using Cisco 3725s

What it demonstrates - eBGP session, default route advertisement, local preference and failover

Why it matters - Shows real internet routing behavior 


Lessons learned: 
Can only use routed ports; cannot use any NM-16ESW/VLANs in Dynamips
